Result 3: Interactive Learning Tool

Oct 12, 2021
Follow

LearningML aims to be a free/open-source web-based platform aimed to bring Machine Learning (ML) fundamentals in a practical way to children or people interested in the growing field of AI.

This tool provides a platform intended to build ML models from example data, and a Programming Platform allowing to code creative applications using those models. LearningML has been created by the spanish teacher Juan David Rodríguez García (http://juandarodriguez.es) who is also currently developing and maintaining the project. The collaborator list and their contributions can be found in https://web.learningml.org/agradecimientos/
